上海磐云电子科技有限公司简称（磐云科技），是一家（多媒体）系统专业应用的解决方案提供商。同时它也是集工业级大屏幕显示系统产品的研发、制造、销售及服务于一体的高新技术企业。自成立起就一直致力中国的专业（多媒体）市场，代理、经销世界知名品牌，以专业的系统集成及优质的增值服务，为客户提供专业、完善的系统解决方案。作为该领域高品质系统设备的一流供应商、服务商及开发生产商，还广泛开展与国内外制造商合作，并与部分国内外厂商都建立了良好的战略合作关系。 磐云科技还与众多的代理商或生产商建立了长期的战略合作伙伴关系。在前期用户方案设计投标、工程安装、售后响应服务等方面给予合作伙伴全方位支持，通过大量的代理商和集成商把磐云科技的产品及服务送到广大客户手中，双方协作配合，共同达到双赢的营销局面。 展望未来，磐云科技将本着为客户提供专业解决方案和应用服务的方针，来满足客户各层次的需求，令产品和服务更完美。 磐云科技真诚的期待与您的合作。
